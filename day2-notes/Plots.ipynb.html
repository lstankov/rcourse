Introduction

Graphics in R is far too large a topic to cover in a course like this. As such I have decided to keep it both very simple and very limited.

My statistical philosophy is that plotting of data should be central to all analysis from data screening, to exploration, to final product. In many ways visual display of information has the ability to better represent both trend and noise in a way that statistics cannot.

As I simple do not have the capacity to go through plotting in anywhere near sufficent detail, I detail the following links:

  1. Grammar of graphs as instigated in R via ggplots
  2. Classic works by Edward Tufte
  3. For beautiful graphs try flowing data
  4. For colour pallets go here or create your own [here](http://app.coolors.co/517561-629c7c-3b5d49-262f2a-324b3a] and use the hex values in R e.g., col = "#D7E8D5"
  5. To see the possibilities go to plotly
  6. To just get stuff done try the R cookbook
  7. For inspiration on interactive graphs go to gapminder and produce the graphs in R with googleVis or shiny
  8. Real time animated graphs are also possible
  9. Be a hipster and use Wes Anderson colour pallets
  10. Animations in R

Top Tips

  1. Know the comands for plot, abline, line, hist, density, and curve to quickly generate graphs for exploration and production
  2. pairs and corrplot will let you look at lots of pairwise relationships quickly
  3. Start simply with default plot settings and slowly build up
  4. For likert data use jitter to move points around a bit to better see relationships
  5. For large data use alpha blending to make points semi transperent to better see where the density of points is (see alpha or rgb functions)
  6. For big data use heat maps, ggobi, bigvis, or hexbin plots
  7. Learn ggplots as this seems to be where everyone is heading
  8. Many many functions have default plots so try throwing output from a function at plot(). For models this will often be plots of assumptions
  9. Follow the general rule of avoiding chartjunk (Tufte, 1983 - Above all else show the data)
  10. Know what devices are best to export to (you might want to export as an svg and do further work in illustrator or inkscape)
  11. Know the big names and get their books, blogs, t-shirts (Edward Tufte, Stephen Few, Hadley Wickham).

Scatter plots, box plots, and Histograms

Here I give examples of moving from default graphs to prettier graphs.

%%R
#Scatter Plot - Ugly
with(pisa, plot(SCMAT, ANXMAT))

# Scatter Plot - Pretty
with(pisa, plot(jitter(SCMAT,amount = 2), jitter(ANXMAT,amount = 2), pch = ".", cex = 3, col = "#00000075",
				main = "Scatter Plot of\nSelf-concept and Anxiety", xlab = "Self-concept", ylab = "Anxiety",
				bty="n"))

#Lets add a regression line
abline(lm(pisa$ANXMAT ~ pisa$SCMAT), col = rgb(1,0,0,.6), cex = 2)

#Histogram - Ugly
hist(pisa$ESCS)
#Histogram - Pretty
hist(pisa$ESCS,freq = FALSE, col = "grey", border = "grey50")
lines(density(pisa$ESCS, na.rm=TRUE), col = "red")

#Bar plot - ugly
boxplot(SCMAT ~ IMMIG, data = pisa, main = "Boxplot of self-concept by \nImmigrant Status")
#Bar plot - pretty
boxplot(SCMAT ~ IMMIG, data = pisa, main = "Boxplot of self-concept by \nImmigrant Status",
		notch = TRUE, col = c("grey25", "grey50", "grey75") )

GGPLOTS

SPSS has, for sometime, tried to use a sort of grammar of graphs logic to their plots. However, SPSS does not really have this sort of flexibility required (at the very least I find it annoying). One of the main people in this framework, Hadley Wickham, however has made a grammar of graphs package for R called ggplots 2. I rarely use it as I now used to base graphics BUT ggplots are powerful and beautiful. Below I create a pretty scatter plot and full information bar chart (i.e. every data point is shown).

The idea behind ggplots is to build graphs in discreet stages or components seperated by '+'. ggplots also has a powerful faceting system in which multiple graphs are presented in rows and columns to provide a rich but readable picture of your data.

In [14]:
%%R
#Scatter plot
library(ggplot2)
print(ggplot(pisa, aes(SCMAT, ANXMAT, color = as.factor(yearInSchool)) ) + 
	geom_point(size = 3, position=position_jitter(width=1,height=.5), alpha = .6) + 
	scale_color_manual(values = wes_palette("GrandBudapest")) +
	scale_x_continuous(name = "Anxiety") + scale_y_continuous(name = "Self-concept") + 
	ggtitle("Self-concept by Anxiety") + 
	theme(plot.title=element_text(family="Times", face="bold", size=20)) +
	guides(color=guide_legend(title="Year in school")) )
# Fancy bar plot
print(ggplot(pisa, aes(as.factor(yearInSchool),SCMAT) )+
	geom_point(position = position_jitter(width=.2,height=.5), alpha = 0.6) +
	scale_x_discrete(breaks= 8:11, labels=c("Yr8", "Yr9", "Yr10", "Yr11"), name = "Year in school") +
	scale_y_continuous(name="Self-concept") +
	ggtitle("Self-concept by year in school") + 
	theme(plot.title=element_text(family="Times", face="bold", size=20)) )
